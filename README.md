## ğŸš€ Getting Started

### Prerequisites
- Node.js 18+ 
- pnpm (recommended) or npm

### Installation

1. **Clone the repository**
   ```bash
   git clone [your-repo-url]
   cd samba-shampoo-landing
   ```

2. **Install dependencies**
   ```bash

   ```
